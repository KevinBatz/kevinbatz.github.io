# Kevin Batz Website

Static GitHub Pages site for Kevin Batz.

## Deploy

1. Create a GitHub repository named `kevinbatz.github.io`.
2. Copy these files into the repository root.
3. Commit and push to the `main` branch.
4. In GitHub, open Settings > Pages and select deployment from the `main` branch root if it is not enabled automatically.

The site has no build step and no external dependencies.
